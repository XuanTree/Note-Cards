# English

## Note Cards

A simple extension to let you view all your md notes as cards! If you have used note-taking apps like [Obsidian](https://obsidian.md/), you will find using this extension easy.

### Features

1. ✅View all your md notes as cards.
2. ✅Open your note directly from the cards.
3. ✅Filter your notes by tags.
4. ✅Add a Cover to your cards. 
5. ✅Light/Dark mode can be chosen!

### Usage

1. Open VSCode.
2. Click the extension market
3. Search for "Note Cards" and install it.
4. Open a folder containing your md notes.

You can use Yaml front matter in your md notes to add a cover to your cards. The cover will be displayed on the card. The cover can be a link to an image or a local image.**Using command `Ctrl+Shift+P` to open the command palette and type "Note Cards: Open View" to open the panel.**

# 中文

## Note Cards

一个简单的扩展，让你可以将所有的md笔记以卡片的形式查看！如果你使用过像[Obsidian](https://obsidian.md/)这样的笔记应用，你会发现使用这个扩展非常简单。

### 功能

1. ✅将所有的md笔记以卡片的形式查看。
2. ✅直接从卡片中打开你的笔记。
3. ✅通过标签过滤你的笔记。
4. ✅为你的卡片添加封面。
5. ✅可以选择浅色/深色模式！

### 使用方法

1. 打开VSCode。
2. 点击扩展市场
3. 搜索"Note Cards"并安装它。

4. 打开一个包含你的md笔记的文件夹。

你可以在你的md笔记中使用Yaml front matter来为你的卡片添加封面。封面可以是一个链接到图片的链接，也可以是一个本地图片。**使用命令`Ctrl+Shift+P`打开命令面板，输入"Note Cards: Open View"来打开面板。**
